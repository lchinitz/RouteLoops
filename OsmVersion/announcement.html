RouteLoops Announcement

RouteLoops has been creating closed-loop routes for many years,
and to all of our users over that time we say "thank you."

End of availability of the Google Maps based version

The primary RouteLoops.com site uses Google Maps, and Google Maps APIs, to generate and present routes. As that has become more and more expensive, and since we provide the RouteLoops service for free, we have decided to no longer offer the Google-Maps-based interface.

Continued support of the OSM based version

There are other versions of RouteLoops built on top of other mapping services (Open Street Maps and MapBox), and we have chosen to continue offering the free RouteLoops service built on OSM.

Other options to get RouteLoops

RouteLoops has been fortunate, over the years, to work with several companies interested in offering closed-loop routing for outdoor activities. Among the best of these has been Garmin, which has built RouteLoops into their Round-Trip Routing offering. Many Garmin devices support Round Trip Routing, and with an account you can build Round-Trip Routes inside of Garmin Connect.

Announcing the availability of the RouteLoops code

Although we have chosen not to continue providing the Google Maps version of RouteLoops, with the expiration of the RouteLoops patent (7,162,363) we have decided to put the RouteLoops code into the public domain.
Working examples of the RouteLoops code are now available on github. There are three versions of the code:
  1. Google Maps version
  2. Open Street Maps version using Leaflet
  3. MapBox version
So if you really want the version running on Google Maps, feel free to download the code and run it locally. And if anyone would like to create a new site using either the concept, or the code, you are welcome to do that. If you do, please inform the wider RouteLoops community.
